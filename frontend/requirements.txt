# Frontend Requirements for OMANI Therapist React UI
# This file lists all necessary dependencies for the React frontend

# Core Requirements
# ================

# Node.js - JavaScript runtime
# Minimum version: 16.0.0
# Recommended version: 18.0.0 or higher
# Installation: Download from https://nodejs.org/

# npm - Package manager
# Minimum version: 8.0.0
# Comes with Node.js installation

# System Requirements
# ==================

# Operating System:
# - Windows 10/11
# - macOS 10.15+
# - Linux (Ubuntu 18.04+, CentOS 7+)

# Browser Support:
# - Chrome 88+
# - Firefox 85+
# - Safari 14+
# - Edge 88+

# Hardware Requirements:
# - RAM: 4GB minimum, 8GB recommended
# - Storage: 2GB free space
# - CPU: Modern multi-core processor

# Development Tools (Optional but Recommended)
# ==========================================

# Code Editor:
# - Visual Studio Code with extensions:
#   - ES7+ React/Redux/React-Native snippets
#   - TypeScript Importer
#   - Prettier - Code formatter
#   - ESLint
#   - Material-UI Snippets

# Git - Version control
# Installation: https://git-scm.com/

# Installation Instructions
# ========================

# 1. Install Node.js from https://nodejs.org/
# 2. Verify installation:
#    node --version
#    npm --version
# 3. Navigate to the frontend directory:
#    cd frontend/react-app
# 4. Install dependencies:
#    npm install
# 5. Start development server:
#    npm start

# Package Dependencies
# ===================
# All React dependencies are managed through package.json
# Run 'npm install' to install all required packages

# Environment Variables Required
# =============================
# Create a .env file in frontend/react-app/ with:
# REACT_APP_API_URL=http://localhost:8000
# REACT_APP_VERSION=1.0.0
# REACT_APP_ENVIRONMENT=development

# Additional Setup
# ===============
# - Ensure backend API is running on port 8000
# - Configure CORS in backend for port 3000
# - Set up proper API keys in backend .env file
